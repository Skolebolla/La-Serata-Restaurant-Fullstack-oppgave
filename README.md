Dette prosjektet går ut på å utvikle en fullstack-løsning for en fiktiv restaurant, La Serata, som inkluderer en nettside der man kan utforske menyen og/eller bestille bord. Menyen ligger i en database, det samme gjelder bord-bookinger. Det er også en mange-til-mange relasjon med menyen for allergener som finnes i maten.

Nettsiden er laget med HTML, CSS og JavaScript.
Databasene er laget med postgreSQL, og docker er tatt i bruk.
